# EduChain

A decentralized educational achievement platform with skill-based rewards on Stacks blockchain.

## Features

- Learning achievement recording and skill progression tracking
- Achievement point calculation based on educational milestones
- Academic dean oversight for curriculum management and validation
- Honor-based graduation rewards for outstanding academic performance
- Transparent knowledge bank for collaborative learning advancement

## Smart Contract Functions

### Public Functions
- `establish-academy` - Initialize educational platform
- `record-achievement` - Record learning achievements
- `calculate-achievement-points` - Calculate points (dean only)
- `graduate-with-honors` - Graduate and claim honor rewards

### Read-Only Functions
- `get-student-achievements` - Get student's achievements
- `get-academy-stats` - Get platform statistics
- `get-knowledge-bank` - Get total knowledge bank

## Usage

Deploy the contract to create a decentralized educational platform where students can record their achievements and earn honor rewards based on their learning contributions and academic progress.
